# MetalMind AI — Landing Page

Marketing landing page for **MetalMind AI**, an LLM-native import optimisation platform built at an AI Hackathon.

**Live demo:** [metalmindproject-production-1c44.up.railway.app](https://metalmindproject-production-1c44.up.railway.app/)

---

## About

MetalMind AI analyses global prices, supplier data, and logistics costs in real time — helping procurement teams make faster, more cost-effective import decisions with confidence.

Key problems it solves:
- Price opacity across global markets
- Fragmented supplier evaluation
- Compliance & risk blind spots
- Slow, manual decision cycles

---

## Tech stack

- Single-file static site (`index.html`) — HTML, CSS, and JS all inline
- No build system or package manager
- Fonts via Google Fonts: `DM Sans`, `Playfair Display`, `JetBrains Mono`

---

## Running locally

Open `index.html` directly in a browser, or serve with any static server:

```bash
python -m http.server
```
